# 手语助手 - 无障碍辅助应用

一款专为听障人群设计的iOS风格移动应用，提供全方位的沟通和信息获取支持。

## 核心功能

### 1. 视觉文字辅助（OCR识别）
- 📸 **拍照识别**：通过相机实时拍摄并识别文字内容
- 🖼️ **相册选择**：从相册中选择图片进行文字识别
- 🔍 **文字放大**：支持识别结果的文字放大查看
- 📋 **快速操作**：复制、分享识别结果
- 📚 **历史记录**：保存识别历史，方便回顾

**模拟技术栈**：OPPO OCR SDK 集成

### 2. 手语交互
- ✍️ **文字转手语**：输入文字自动转换为手语图片序列
- 🤝 **手语转文字**：通过摄像头实时识别手语动作并转换为文字
- 💬 **常用短语**：提供常用手语短语快速选择
- 🎬 **动画演示**：手语动作的动画演示功能
- 📖 **学习模式**：帮助用户学习和掌握手语

**模拟技术栈**：MediaPipe + LSTM 手语识别引擎

### 3. 环境音感知
- 🔔 **多类型检测**：识别门铃、警报、电话、婴儿哭声、狗叫、敲门声等
- 📊 **灵敏度调节**：支持自定义检测灵敏度（0-100%）
- 📳 **多种提醒方式**：
  - 震动提醒
  - 屏幕闪烁
  - 推送通知
- 📈 **统计分析**：提供检测次数、准确率等数据统计
- 🕐 **检测历史**：记录所有检测事件

**模拟技术栈**：CNN 声音识别模型

### 4. 社区论坛
- ✏️ **发帖功能**：分享经验、提问互助
- 💬 **评论互动**：支持点赞、评论、分享
- 👥 **关注系统**：关注感兴趣的用户
- ⭐ **收藏功能**：收藏重要帖子
- 🎯 **内容分类**：推荐、关注、收藏三个栏目

### 5. 个人中心
- 👤 **个人资料**：管理用户信息
- 📊 **数据统计**：帖子数、关注数、粉丝数
- 🎖️ **积分成就**：使用天数、积分、成就展示
- ⚙️ **设置选项**：
  - 通知设置
  - 震动提醒
  - 隐私安全
  - 帮助中心

## 使用指南

### 首次使用
1. 打开应用会显示欢迎引导页
2. 浏览各功能介绍
3. 点击"立即开始"进入应用
4. 可在登录页面注册账号（目前为演示模式）

### 文字识别
1. 进入首页，点击"拍照识别"或"相册选择"
2. 拍摄或选择包含文字的图片
3. 系统自动识别并显示文字内容
4. 可复制、分享或保存结果

### 手语转换
1. 切换到"手语"标签
2. **文字转手语**：输入文字，点击转换查看手语图片
3. **手语转文字**：点击"开始识别"，对着摄像头做手语动作

### 环境音监听
1. 切换到"声音"标签
2. 选择要监听的声音类型
3. 调整识别灵敏度
4. 点击"开始监听"

### 社区互动
1. 切换到"社区"标签
2. 浏览推荐、关注的帖子
3. 点击右上角"+"发布新帖
4. 可以点赞、评论、收藏帖子
